The <head> element

The <head> element is a container for metadata placed between the <html> tag and the <body> tag.

Metadata define the document title, character set, styles, scripts and other meta information.

This elements can go inside the <head> element:

  • <title>
  • <style>
  • <base>
  • <link>
  • <meta>
  • <script>
  • <noscript>
