CSS
(Cascading Style Sheet)

This is CSS:

Selector{ property:value}

AND this is pretty much it.

(click selector , property or value)

To insert CSS We have three ways:

  • Internal CSS
  • External CSS
  • Inline CSS

Internal CSS

Style sheet you can place into <style> into <head> section

HTML
(index.html)


<!DOCTYPE html>
<html>
<head>
<title>Internal CSS</title>
<style type="text/css">
body{
background:grey;
}
h1{
color:red;
}
p{
color:blue;
}
</style>
</head>
<body>
<h1>Internal CSS</h1>
<p>Color of background is GREY, heading is RED and paragraph is BLUE</p>
</body>
</html>





External CSS

Style sheet you can reference to the external style sheet file inside the <link> element, inside the <head> section.

HTML
(index.html)


<!DOCTYPE html >
<html>
<head>
<title>External CSS</title>
<link rel="stylesheet" type="text/css" href="style_external.css">
</head>
<body>
<h1>External CSS</h1>
<p>Color of background is GREY, heading is RED and paragraph is BLUE</p>
</body>
</html>

CSS
(style_external.css)


body{
background:grey;
}

h1{
color:red;
}

p{
color:blue;
}




Inline CSS

And style sheet you can put inside the any element, inside/and the <body> section.

HTML
(index.html)


<!DOCTYPE html>
<html>
<head>
<title>Inline CSS</title>
</head>
<body style="background: grey;">
<h1 style="color:red; ">Inline CSS</h1>
<p style="color: blue;">Color of background is GREY, heading is RED and paragraph is BLUE</p>
</body>
</html>




Internal, External or Inline CSS you get identical page.
