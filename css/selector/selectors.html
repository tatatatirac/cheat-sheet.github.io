Selectors

#id Selects the element with id="idName". In CSS file we put id selector like this #idName
.class Selects all elements with class="className". In CSS file we put class selector like this.className
* With * we selects all elements
element Selects all elements in added property like p, h1, body, div, span etc.
element, element... Selects 2 or more elements in document
