CSS #id Selector

In CSS file:

#thisIsIDSelector{

      color:green;

  }





In HTML file:

<!DOCTYPE html>
<html>
<head>
   <title>Some title</title>
</head>

  <body>
    <h1 id="thisIsIDSelector" >Some text</h1>
  </body>

</html>


COLOR is property, GREEN is a value.

Also you can write in <style> element, into <head> element
